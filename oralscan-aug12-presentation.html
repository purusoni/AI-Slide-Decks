The Challenge: Stagnation in Dental AI

Performance Plateau

Current models like YOLOv12x are stuck at ~0.37 mAP, limiting clinical utility.

Data Imbalance

Severe class imbalance, with rare pathologies like lesions at <1% of the dataset.

Feature Ambiguity

Subtle pathological features are difficult for standard models to distinguish.

A Four-Pronged Strategic Solution

  • Data-Centric Advancement: Hybrid SSL on unlabeled data.
  • Architectural Innovation: Swin Transformer + DINO-DETR.
  • Advanced Training: Specialized loss functions for fine-tuning.
  • Diagnostic-Driven Refinement: XAI and clinical feedback loops.

Technical Architecture

Phase 1: SSL Backbone

Swin Transformer (28M params) trained with a novel Hybrid SSL Framework (InfoNCE + Barlow Twins + MAE) to learn robust general features.

Phase 2: Detection Head

DINO-DETR Framework for end-to-end object detection, eliminating complex post-processing like NMS and improving localization.
